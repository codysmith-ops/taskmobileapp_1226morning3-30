Design System

Complete visual language and component library for Ellio mobile app

üé® Critical Design Rules

  • NO EMOJIS - Always use SVG icons from this system
  • Use #1E293B for dark backgrounds (NOT pure black)
  • Alert colors (#DC2626, #D97706, #059669) are ONLY for notifications and status indicators
  • All borders are 1px only - never 2px or 3px
  • Icon stroke width: 2.5-3px for consistency
  • Labels on dark backgrounds always use #818CF8

Property Value Usage
