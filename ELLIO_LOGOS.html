Logo Usage Guidelines

  • Primary Color: #5159B0 (Brand Indigo) - Use on light backgrounds
  • Dark Mode: #818CF8 (Light Indigo) - Use on dark backgrounds (#1E293B)
  • Clear Space: Maintain minimum padding of 20px around all logos
  • Minimum Size: Icon: 24px, Wordmark: 80px wide, Full Logo: 140px wide
  • App Icon: 1024x1024px master, with 26px corner radius for iOS
  • Stroke Width: All icon elements use 2.5-3px stroke width for consistency
  • Typography: Wordmark uses Montserrat ExtraBold (800 weight)
